## 🆘 If Something Goes Wrong

### Problem: Backend won't start
```bash
docker-compose logs backend
# Look for error messages
# Common: Missing .env file or port already in use
```

### Problem: "GEMINI_API_KEY not found"
```bash
# Verify .env exists and has the key
ls -la .env
grep GEMINI_API_KEY .env

# Restart with explicit env file
docker-compose --env-file .env -f docker-compose.dev.yml restart backend
```

### Problem: API key error
```bash
# Test the key works
python << 'EOF'
import google.generativeai as genai
genai.configure(api_key="<YOUR_GEMINI_API_KEY>")
model = genai.GenerativeModel("gemini-pro")
print(model.generate_content("Hello"))
EOF

# If error, regenerate key at: https://makersuite.google.com/app/apikey
```
